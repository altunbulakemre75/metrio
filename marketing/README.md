## GitHub Pages yayınlama

1. Repository ayarları → Pages → Source: "Deploy from a branch"
2. Branch: `main`, folder: `/site`
3. Kaydet, URL'i bekle: `https://{kullanici}.github.io/{repo}/`

Ya da alternatif: **Carrd.co** (ücretsiz, 5 dk kurulum) → `index.html` içeriğini kopyala-yapıştır.
